### List of Features

- Sign in and Register 
  - Sign in with Google supported
  - Forgot password feature
  - Email has to be unique

- Home Screen
  - Any books recently added appear in section "Recently Rated"
  - A small list of suggested material based on users Reading Log is presented under the section "Recommended for You"
  - "Quote of the Day" - this presents a different quote every 24 hours

- Search Screen
  - Ability to search for books by title, author, genre or by scanning the ISBN barcode

- Review of Books
  - When reviewing a rating from the Open Library is shown
  - Ability to add to future reads
  - Ability to Star a book if it is a favourite
  - Review score and comments can be added
  - The book format can be added as well
  - Reviews can be edited or deleted at any time

- Reading Log Screen
  - List of books logged by user. Shows rating and any review comments. Other user reactions to review comments are also shown here
  - Total number of books logged shown at top
  - Ability to "Sort" by date (default), alphabetical, rating, author

- Profile Screen
  - Avatar profile picture
  - Achievements section
  - Friends
  - Activity Sctream
  - Stats breakdown: Total books completed (further details breakdown by book type e.g. audio), top authors, top rated books, stats by year

- Friends Feautre
  - Shows list of friends
  - Ability to search by friends username
  - Any friend requests shown here
  - Ability to look at friends (or public user) account → total books read, achievements, users reading log, stats breakdown

- Future Reads Screen
  - Users can mark books for a future read and view the list here
  - Any book in this list can be reviewed straight from here

- Activity Stream
  - Any review comment reacted to will show here

- Authentication
  - Leveraging Firebase

- Data Storage and Persistence
  - Leveraging Firebase

- Account Setting
  - Ability to change users password
  - Ability to change users visible username
  - Ability to modify the Profile Privacy setting (Public, Friends Only, Private)
  - Ablity to change the app theme (Default, High Contrast, Batman, Dark Autumnal)
  - User can also sign out and delete account here
